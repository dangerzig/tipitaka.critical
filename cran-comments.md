## New submission

This is a new package providing a lemmatized critical edition of the
complete Pali Canon (Tipitaka). It is a companion to the existing CRAN
package 'tipitaka', which provides the original VRI edition.

The package ships a single text dataset (4.2 MB tarball) and computes
derived frequency tables and sparse matrices on demand.
